# NeoPOS(BETA v0.1) 

Aplikasi Kasir Online berbasis Web yang modern, responsif, dan terintegrasi Cloud Database.

##  Fitur Unggulan
- **Multi-Device Sync:** Data sinkron otomatis antara HP, Tablet, dan Laptop.
- **Manajemen Stok:** Tambah, Edit, dan Hapus produk dengan mudah.
- **Laporan Keuangan:** Grafik omzet harian/bulanan dan riwayat transaksi detail.
- **Cetak Struk:** Mendukung printer thermal bluetooth/USB.
- **Pembayaran:** Mendukung Tunai & QRIS dengan kalkulasi kembalian otomatis.

##  Teknologi
- **Frontend:** HTML, Tailwind CSS, Vanilla JS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Neon Tech)
- **Deployment:** Vercel

##  Cara Install (Lokal)
1. Clone repo ini.
2. `npm install`
3. Buat file `.env` dan isi `DATABASE_URL=link_neon_anda`
4. `npm start`
